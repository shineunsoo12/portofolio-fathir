About Me & Kenapa Saya Masuk TKJ

Halo! Saya adalah seorang siswa SMK yang memiliki semangat tinggi dalam dunia teknologi, khususnya di bidang Teknik Komputer dan Jaringan (TKJ). Memilih jurusan TKJ bukanlah keputusan yang saya buat secara asal-asalan.

Dunia saat ini bergerak sangat cepat menuju era digitalisasi di mana segala sesuatu saling terhubung melalui internet. Saya sangat tertarik untuk memahami bagaimana data dapat dikirim dari satu titik ke titik lain di seluruh dunia dalam hitungan detik, bagaimana infrastruktur jaringan dibangun agar stabil dan aman, serta bagaimana mengatasi masalah (troubleshooting) yang kompleks.

Di SMK PLUS PELITA NUSANTARA, saya belajar tidak hanya teori, tapi juga praktek langsung mulai dari konfigurasi router, instalasi server, hingga fiber optic. Saya melihat peluang karir yang sangat luas di bidang ini sebagai Network Engineer atau IT Specialist. TKJ memberikan saya fondasi yang kuat untuk memahami "otak" dari teknologi modern, dan itulah alasan utama saya mendalami jurusan ini dengan sungguh-sungguh.

Keahlian Saya

Mikrotik

Saya memiliki kemampuan dalam mengonfigurasi perangkat Mikrotik untuk berbagai kebutuhan jaringan. Mulai dari pengaturan dasar seperti Winbox, manajemen bandwidth (Simple Queue dan PCQ), setting hotspot voucher, hingga routing dasar. Saya memahami bagaimana mengamankan jaringan dan membagi koneksi internet secara adil untuk pengguna.

Fiber Optic

Teknologi Fiber Optic adalah tulang punggung internet modern. Saya belajar teknik splicing (penyambungan) kabel serat optik dengan presisi tinggi menggunakan Fusion Splicer. Selain itu, saya juga memahami cara instalasi ODP (Optical Distribution Point) dan penanganan kabel agar tidak mengalami bending loss yang berlebihan, serta pengujian sinyal menggunakan OTDR atau Power Meter.

Server

Server adalah jantung dari sebuah jaringan. Saya memiliki kompetensi dasar dalam merakit dan mengonfigurasi server fisik maupun virtual. Keahlian ini mencakup instalasi sistem operasi server (seperti Debian/Ubuntu atau Windows Server), konfigurasi web server (Apache/Nginx), DNS Server, dan DHCP Server untuk memastikan layanan jaringan berjalan lancar dan tersedia bagi klien.
